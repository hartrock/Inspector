URLs
http://localhost:8080/symbols.html#MAIN jump to MAIN context folder
http://localhost:8080/symbols.html#Class:Class jump to Class:Class symbol inside Class context folder
http://localhost:8080/symbols.html#FOOPReference:delete-all-refs jump to FOOPReference:delete-all-refs symbol inside FOOPReference context folder
http://localhost:8080/symbols.html#Help jump to Help
Special OPs
http://localhost:8080/leave leave Inspector server process (to be (re)loaded three times to end loop of demo)
http://localhost:8080/symbols-JSON show all symbols of all contexts in JSON format
http://localhost:8080/symbols-JSON?inList=Class:Class show all symbols referenced by some lambda/macro/list (here Class:Class) in JSON format
