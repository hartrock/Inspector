Help

URLs
symbols.html#MAIN jump to MAIN context folder
symbols.html#Class:Class jump to Class:Class symbol inside Class context folder
symbols.html#FOOPReference:delete-all-refs jump to FOOPReference:delete-all-refs symbol inside FOOPReference context folder
symbols.html#Help jump to Help
symbols.html?file=/tmp/snapshot.json load symbols from a snapshot file (a jump hash as URL suffix is possible, too)
Special OPs
leave leave Inspector server process (to be (re)loaded three times to finish startIt_loop.lsp loop)
symbols.html?pingPong ping-pong between Inspector server process and browser GUI (see below)
symbols-JSON show all symbols of all contexts in JSON format
symbols-JSON?inList=Class:Class show all symbols referenced by some lambda/macro/list (here Class:Class) in JSON format
Ping-pong mode
Just run
  ./startIt_pingPong.lsp
: in terminal output there is info about how to proceed.
